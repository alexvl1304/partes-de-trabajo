# Partes de trabajo

Aplicación web progresiva para realizar partes de trabajo de forma online y offline.

# Instalación

-Clonar el repositorio en tu equipo. 

-Ejecutar desde power shell en la ubicación del proyecto 'npm install'

# Ejecución

-Consultar tu IPv4 con 'ip config'

-Ejecutar el comando 'npm run dev -- --host', te saldrá el puerto en el que está corriendo la ejecución.

# Para entrar en la aplicación hay dos opciones:

1. Desde el navegador de tu equipo entrar en localhost: + puerto anterior.

2. Desde el navegador de otro dispositivo conectado a la misma red entrar a la ip: + puerto anterior.
